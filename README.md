# Task Management API

REST API built with Express, Sequelize and MySQL.

## Features

- User registration
- User ↔ Task relation (1:N)
- Filtering
- Pagination
- Sorting
- Status workflow
- Priority levels

## Tech Stack

- Express
- Sequelize
- MySQL
- TypeScript
